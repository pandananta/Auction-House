=== Auction House ===
Ananta Pandey, 2013

Auction House is the backend for an item Auctioning program. 

== Description ==
The object files included form the backend of the Auction House program, 
which allows you to start an auction for an item with a minimum price, and bid until it's over.
To demonstrate the functionality, the Auction House class run a series of unit tests.

To compile, be sure to include junit-4.11.jar and hamcrest-core-1.3.jar in your class/build path.
